Introduction to ACEBounds

Simulate Data

To use for illustrative purposes, we simulate some data. The DAG corresponding to the model simulated from is presented below. \(X,Y,U\) are all binary, while \(Z1\) is categorical with three levels.

set.seed(184506)

library(ACEBounds)
library(magrittr)
library(dplyr)

sim_dat <- simulate_data(sample_size = 500,
                         IVs_ps = list(c(1, 1, 1)/3),
                         indIVs_on_X = 0.7)

library(ggdag)
plot_DAG(sim_dat) + scale_x_reverse()

With Individual Level Data

We create three data sets to illustrate the scenarios where either trivariate or bivariate data is available.

XYZ_data <- sim_dat$simulated_data

XZ_data <- sim_dat$simulated_data %>% select(X,Z1)
YZ_data <- sim_dat$simulated_data %>% select(Y,Z1)

Trivariate data source

Bivariate data sources

## [1] "bivariate"
Y_probs <- probs_from_data(YZ_data, Y = Y, Z = Z1)

get_bounds(thetas = X_probs$thetas,

With Summary Data Only

If you do not have individual level data available, you need to either provide vectors of \(P(X = 1 | Z = z)\) and \(P(Y = 1 | Z = z)\), or an array of \(P(X = x, Y = y | Z = z)\).

In the trivariate case, we take as an example the Vitamin A supplementation data from (???).

counts <- array(data = c(74, 0, 11514, 0,
                         34, 12, 2385, 9663),
                dim = c(2,2,2),
                dimnames = list(x = 0:1,
                                y = 0:1, 
                                z = 0:1))

pander::pandoc.table(counts, style = "rmarkdown")
z 0 1
x y
0 0 74 34
1 11514 2385
1 0 0 12
1 0 9663

We convert the counts to the respective proportions.

zetas <- counts / rep(apply(counts, 3, sum), each = 4)

pander::pandoc.table(zetas, style = "rmarkdown")
z 0 1
x y
0 0 0.006386 0.002811
1 0.9936 0.1972
1 0 0 0.0009922
1 0 0.799
